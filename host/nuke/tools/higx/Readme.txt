Instalation:
====================================
1)	- Place the /higx/ folder with all of it's content into your ~/.nuke folder.
2.0)	- Open your ~/.nuke/init.py in a text editor
2.1)	- Add a new line to the end of the file with the following text:

nuke.pluginAddPath('./higx/PointRender')

2.2)	- Save and close the init.py file
3)	- When you open nuke you should now see a new Higx icon in your tool row containing the tools



.nuke default folder location
Linux:       /home/login name/.nuke
Mac OS X:    /Users/login name/.nuke
Windows:     drive letter:\Users\user name\.nuke

for more info https://support.foundry.com/hc/en-us/articles/207271649-Q100048-Nuke-Directory-Locations
